# Kubernetes_Notes_DevOps
Core Concepts, Workloads, Networking, Storage, Scaling and Scheduling, Cluster Administration, Monitoring and Logging and Security

Topics
1. Core Concepts
Monolithic Vs Microservices, Kubernetes Architecture, Set up on local/AWS EC2, Kubectl, Pods, Namespaces, Labels, Selectors, Annotations

2. Workloads
Deployments, Statefull sets, Daemon sets, Replica sets, Jobs, Cron jobs

3. Networking
Cluster networking, Services, Ingress, Network Policies

4. Storage
Persistent Volumes (PV), Persistent Volume Claims (PVC), StorageClasses, ConfigMaps, Secrets

5. Scaling and Scheduling
HPA (Horizontal Persistent Autoscaling, VPA (Vertical Persistent Autoscaling), Node Affinity, Taints/Tolerations, Resource Quotas, Limits, Probes

6. Cluster Administration
RBAC, Cluster Upgrade, Custom Resource Definintions (CRD)

7. Monitoring and Logging
Metrics server, Logging, Monitoring Tools. Advanced Features - Operators, Helm, Service Mesh, Kubernetes API

8. Security
Pod Security Standards (PSS), Image Scanning, Network Policies, Secrets encryption. Cloud Native Kubernetes - Manages Services (EKS, AKS, GKE), Cluster Autoscaler, Spot Preemtible Nodes. Debugging and Troubleshooting - kubectl Debugging, Logs, Resource Usage Analysis

9. Projects 
CI/CD Integration: Kubernetes with Jenkins CI/CD, GitOps with ArgoCD, Microservices with MongoDB, .NET, Python Thre-tier App, Montoring with Prometheus, Grafana
